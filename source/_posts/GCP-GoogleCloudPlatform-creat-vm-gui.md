# 建立GCP雲端機器

### 1. 網址：https://cloud.google.com/

### 2. 登入 Google 帳號

### 3. 點擊畫面右上角控制台進入 ( 主頁有各種資訊可以查看 )

![ ](images/1.png)

### 4. 選取或建立新機構（ 專案 )

![ ](images/2.jpg)

### 有專案點左上 ▼ 會跑出專案，點名稱選取專案或是右上角建立新專案

![ ](images/3.2.png)

### ( 建立新專案 ) 為你的專案命名，下方編輯可以更改專案 ID ( ID 更改完後就無法再變更 )

![ ](images/26.png)

### 進入專案畫面，主頁專案資訊有一些預設訊息可以看右上自訂可以調整

![ ](images/5.2.png)
![ ](images/28.png)

### 如果想改專案名稱或是刪除專案可以點右上三個點 → 專案設定

![ ](images/29.png)

### 左邊設定可以更改專案名稱和刪除專案

![ ](images/30.png)
![ ](images/31.png)

### 刪除專案會需要30天，刪除要輸入專案 ID

![ ](images/32.png)

***

### 5. 建立 VM

### 點選資訊主頁畫面左上方的按鈕查看 GCP 產品和服務列表

![ ](images/22.1.png)

### 點選工具列清單 → Compute Engine → VM 執行個體 ( 可點選圖釘把自己常用的工具釘選到最上面 )

![ ](images/4.1.png)

### 點選建立，建立 VM

![ ](images/6.1.png)

### 有建立 VM，點選建立執行個體

![ ](images/27.png)

### 輸入名稱（ 名稱必須為小寫字母、數字或連字號，且結尾須為小寫字母或數字 ）、選擇區域與機器類型

![ ](images/7.1.png)

### CPU 跟記憶體有幫你配好可以直接選擇

![ ](images/33.png)

### 也可以調整成自己想要的核心數量、記憶體大小和 GPU 數量來自訂機器類型

![ ](images/8.1.png)

### 點選變更選擇作業系統 image，及開機磁碟容量及類型

![ ](images/9.png)
![ ](images/10.1.png)
![ ](images/23.1.png)

### 設定身分及 API 存取權與防火牆

![ ](images/11.jpg)

### 設定管理、磁碟、網路、SSH 金鑰

![ ](images/12.jpg)

### 設定管理頁籤內容

![ ](images/13.jpg)

### 點選 ＋新增項目後點選建立磁碟建立第二顆磁碟

![ ](images/14.jpg)
![ ](images/15.png)
![ ](images/16.png)

### 在網路介面下點選筆的圖案，預設 IP 是動態 ( 臨時 ) 可以改成固定 ( 靜態 )  IP 才不會下次無法進入 VM ( 臨時重啟後 IP 會更改 )

![ ](images/17.jpg)
![ ](images/18.jpg)

### 設定 SSH 金鑰

![ ](images/19.jpg)

### 設定完後右上拉開會顯示每個月多少金額

![ ](images/21.2.png)

***

### 6. 檢查和建立後修改

### 再次確認步驟的內容是否正確，確認無誤後點選建立

### 建立 VM 後畫面，打勾或是選擇三個點選擇要做的動作 ( 刪除、停止、重設右上角也有同樣動作可以選 )，點 VM 名子可以進入查看詳細資料

![ ](images/20.1.png)

### 點編輯可以再次對 VM 做編輯，但是 CPU、記憶體、image 設定好無法再更改，一樣可以做刪除、停止、重設等動作

![ ](images/24.png)
![ ](images/25.png)
